# HW-Sequelize-burger
UCLA Homework 15

## Link
https://sequelizeburger1000.herokuapp.com/

This a web app, that allows a user to name a burger and it will be saved to mySQL using sequelize.  They can also devour the burger, and it will update the database and move the burger to the already ate column.

## Tecnology Used
* Javascript
* Express
* Express Handlebars
* mySQL
* Sequelize
* HTML
* CSS
